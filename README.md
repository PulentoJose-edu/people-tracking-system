## üë§ Sistema de Detecci√≥n Demogr√°fica (PAR)

### Arquitectura NTQAI

**Modelos BEiT** (Bidirectional Encoder representation from Image Transformers):
- **Backbone**: Microsoft BEiT-base-patch16-224
- **Input**: Bounding box recortado 224x224
- **Batch processing**: Procesa m√∫ltiples personas simult√°neamente
- **Cach√© inteligente**: Evita re-procesar la misma persona

### Categor√≠as de Clasificaci√≥n

**G√©nero:**
- Masculino (M)
- Femenino (F)

**Edad (5 rangos):**
- **0-18 a√±os**: Ni√±os y adolescentes
- **19-35 a√±os**: Adultos j√≥venes
- **36-60 a√±os**: Adultos
- **60+ a√±os**: Adultos mayores
- **Desconocido**: No detectado o baja confianza

### Optimizaciones de Performance

- ‚úÖ **An√°lisis throttled**: Se ejecuta cada 15 frames (configurable)
- ‚úÖ **Cach√© por track_id**: Reutiliza predicciones previas
- ‚úÖ **Lazy loading**: Modelos se cargan solo cuando se necesitan
- ‚úÖ **Activaci√≥n opcional**: Sistema PAR puede deshabilitarse

### M√©tricas de Rendimiento

| Configuraci√≥n | FPS | Overhead |
|--------------|-----|----------|

**Precisi√≥n**:
- G√©nero: ~95% (modelos NTQAI)
